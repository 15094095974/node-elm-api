# 技术栈

nodejs + express + mongodb + mongoose + es6/7


## 项目运行

```

git clone git@github.com:EasyTuan/node-elm-api.git

cd nuxt-elm

npm install

npm run dev

```

### 1.2 开发过程

#### 1.2.1 命令

```sh
# 开发
npm run dev
```

mongodb常用命令

```sh
mongod --config /usr/local/etc/mongod.conf  # 启动
或
mongod --dbpath .                           # 启动

mongo                                       # 进入mongo命令行
```
